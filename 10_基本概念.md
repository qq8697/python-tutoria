## 关键字
不能把关键字用作标识符的名称。
Python 的标准库提供了一个 `keyword` 模块，可以输出当前版本的所有关键字：
```
import keyword
print(keyword.kwlist)
```

## 多行语句
使用反斜杠(`\`)来实现多行语句
```
total = item_one + \
        item_two + \
        item_three
```

在 `[]`, `{}`, 或 `()` 中的多行语句，不需要使用反斜杠(`\`)
```
total = ['item_one', 'item_two', 'item_three',
        'item_four', 'item_five']
```

## 运算符
1. 算数运算符 
	1. 加、减、乘（拼接字符串 `'-' * 10`）、除、取余（`%`）、取整（`//`）、幂（`**`） 
4. 位运算符
	1. 按位与、或、非、异或、左移、右移
2. 比较运算符
	1. 等于、不等于、大于、小于、大于等于、小于等于
