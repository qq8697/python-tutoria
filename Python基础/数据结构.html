# 数据结构（基本数据类型的方法）

# 一、 序列类型数据

序列类型指：

  1. 基本的序列类型：list、tuple、range；
  2. 处理二进制数据的序列： bytes、bytearray、memoryview；
  3. 处理文本的序列： str。

序列类型数据的常用操作：（s 和 t 是相同类型的序列）

  1. x in/not in s 检查成员
  2. s + t 合并序列
  3. s * n 重复序列
  4. s[i:j:k] 切片操作（截取序列部分内容）
  5. len(s)
  6. max/min(s)
  7. s.index(x, i, j) x 在 s 中第一次出现的索引
  8. s.count(x) x 在 s 中出现的次数
  9. 比较 两个相同类型序列可以比较，相比较的元素按字典序比较

# 二、 可变序列类型数据

不可变序列：主要有 tuple 和 str，此外有 range、bytes；
可变序列：主要有 list，此外有 bytearray。

可变序列类型数据的操作：（ s 是可变序列类型数据的实例，t 是任何可迭代对象）

  1. s[i] = x 替换单个元素
  2. s[i:j:k] = t 替换多个元素
  3. s.append(x) 末尾添加元素
  4. s.insert(i, x) 指定索引插入元素
  1. del s[i:j:k] 删除（多个）元素
  2. s.pop(i) 删除指定索引的元素
  3. s.remove(x) 删除（首个）指定元素
  1. s.extend(t) <=> s += t
  2. s *= n 使用 s 的内容重复 n 次来对其进行更新
  1. s.copy() 浅复制
  2. s.clear() 清空元素
  3. s.reverse() 反转元素

# 三、 字符串

字符串常用方法：

  1. str.capitalize() 首字母大写
  2. str.upper()
  3. str.lower()
  4. str.swapcase()
  5. str.title()
  1. str.center(width, fillchar) 返回一个指定的宽度 width 居中的字符串，fillchar 为填充的字符，默认为空格
  2. str.ljust(width[, fillchar])
  3. str.rjust(width,[, fillchar])
  4. str.zfill (width)
  5. str.expandtabs(tabsize=8)
  1. str.strip([chars])
  2. str.lstrip()
  3. str.rstrip()
  1. str.replace(old, new [, max])
  2. str.maketrans()
  3. str.translate(table, deletechars="")
  1. str.encode(encoding='UTF-8',errors='strict') 编码字符串，如果出错默认报一个 ValueError 的异常，除非 errors 指定的是 'ignore' 或者 'replace' 对应 bytes.decode(encoding="utf-8", errors="strict")
  1. str.endswith(suffix, beg=0, end=len(string))
  2. str.startswith(substr, beg=0,end=len(string))
  3. str.find(str, beg=0, end=len(string))
  4. str.rfind(str, beg=0,end=len(string))
  5. str.index()
  6. str.rindex( str, beg=0, end=len(string))
  1. str.isalnum() / str.isalpha() / str.isdigit() / str.isnumeric() / str.isspace() / str.isdecimal() / str.islower() / str.istitle() / str.isupper()
  1. str.join(seq)
  2. str.split(str="", num=string.count(str))
  3. str.splitlines([keepends])

# 四、 列表

创建列表：

  1. 使用 [] 赋值

    l = ['a', 'b']
    1

  2. 使用列表解析式（也叫生成式或推导式）

    l = [x for x in iterable]
    1

  3. 使用构造函数传入可迭代对象

    list(iterable)
    1

除常见和可变序列的操作，还提供以下方法： l.sort(*, key=None, reverse=None) 排序

列表作为数据结构：

  • 栈（后进先出）： append / pop；
  • 队列（先进先出）： insert(0) / pop(0)，从列表开头进行插入和弹出效率很低，因为所有的其他元素都需要移位，要实现队列，使用 collections.deque ，两端都可以快速的插入和弹出。

# 六、 字典

创建字典：

  1. 使用 {} 赋值

    d = {'one': 1, 'two': 2, 'three': 3}
    1

  2. 字典解析式

    d = {x: x for x in (2, 4, 6)}
    1

  3. 元组组成的列表

    d = dict([('one', 1),('two', 2),('three', 3)])
    1

  4. 关键字参数指定

    d = dict(one=1, two=2, three=3)
    1

  5. dict.fromkeys(iterable[,value=None]) iterable 是可迭代对象（字符串、列表、元组、字典）

    d = dict.fromkeys([2, 4, 6], 0)
    1

字典常用操作：

  1. len(dict)
  2. str(dict)
  3. key in /not in d 检查成员
  4. d.clear() 删除所有元素
  5. d.copy() 浅复制
  1. get() 返回指定 key 的值，如果值不存在返回默认值，不会引发 KeyError（d[key] 如果 key 不存在会引发 KeyError）
  2. setdefault() 添加键并设置默认值
  3. update() 更新键值对
  4. pop(key) 删除并返回字典给定 key 对应的值，
  5. popitem() 删除并返回字典中最后一对键值对（有随机的意味，因为字典是无序的）（无序指的不能人为排序）

字典视图是由 dict.keys(), dict.values() 和 dict.items()返回的对象，字典更改时，视图会反应这些更改。

  1. d.items() 以列表返回可遍历的 (key, value) 元组
  2. d.keys()
  3. d.values()
  4. iter(d) 返回以字典的键为元素的迭代器, 是 iter(d.keys()) 的快捷方式。

字典视图常用操作：

  1. len(dictview)
  2. iter(dictview)
  3. x in dictview

遍历字典：

for k,v in d.items()

# 七、 集合

创建集合：

  1. 使用 {} 赋值

    s = {'a', 'b'}
    1

  2. 集合解析式

    s = {x for x in 'ab'}
    1

  3. 传入可迭代对象

    s = set('ab')
    1

集合基本操作：

  1. s.add(x) 添加元素到集合
  2. s.update(x) 添加元素，参数可以是列表、元组、字典等
  3. s.remove(x) 移除元素，如果不存在，报 KeyError 错误
  4. s.discard(x) 移除元素，如果不存在，不会报错
  5. s.pop() 随机删除一个元素（多次执行结果不一样）（pop 方法对集合进行无序的排列，然后删除左侧第一个元素）
  1. x in/not in s 成员测试
  2. len(s) 计算元素个数
  3. s.clear() 清空集合
  4. s.copy() 浅拷贝
  1. s.difference(*other) 返回多个集合的差集
  2. s.intersection(*other) 交集
  3. s.symmetric_difference(*other) 异或
  4. s.union(*other) 并集

集合运算使用操作符的可读性优于使用方法：

print(set('abc') & set('bcd'))
print(set('abc').intersection('cbd'))
