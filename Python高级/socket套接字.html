# socket 简介

socket（套接字），是操作系统提供的用于（不同主机）进程间通信的一种方式，微信、QQ 等服务多是基于 socket 实现的。 socket API 最初是为 Unix 操作系统开发的，但现在几乎所有的操作系统和语言都有其变体存在。

常用的操作系统套接字 API：

  • s = socket(<parameters>)
  • bind(s, <local IP:port>)
  • connect(s, <remote IP:port>)
  • listen(s,...)
  • s2 = accept(s)
  • n = read(s, buffer, n)
  • n = write(s, buffer, n)
  • close(s)
  • shutdown(s, <side>)
  • getsocketopt(s,...)
  • setsocketopt(s,...)

# socket 模块

是用 Python 的面向对象风格对 Unix 系统调用和套接字库接口的直译。函数 socket() 返回一个套接字对象，其方法是对各种套接字系统调用的实现。

# socket() 函数

socket.socket(family=AF_INET, type=SOCK_STREAM, proto=0, fileno=None)

  • family 地址簇。 （AF_INET（默认，用于 Internet 进程间通信）、 AF_INET6、 AF_UNIX（用于同一台机器进程间通信）、 AF_CAN、 AF_PACKET 或 AF_RDS 其中之一）
  • type 套接字类型。（SOCK_STREAM（默认，流式套接字，主要用于 TCP 协议）、 SOCK_DGRAM（数据报套接字，主要用于 UDP 协议）、 SOCK_RAW 或其他 SOCK_ 常量之一）
  • proto 协议号。
  • fileno 从该文件描述符中自动检测 family、 type 和 proto 的值。

此外，socketpair()、 create_connection()、 fromfd()、 fromshare() 也可以创建套接字对象。

# 套接字对象的常用方法

  • bind(address) 将套接字绑定到 address。
  • listen([backlog]) 启动一个服务器用于接受连接。
  • accept() 接受一个连接，此 scoket 必须绑定到一个地址上并且监听连接。返回值是 (conn, address) 元组，conn 是一个新的套接字对象，用于在此连接上收发数据。
  • connect(address) 连接远程套接字。
  • close() 将套接字标记为关闭。
  • recv(bufsize[, flags]) 从套接字接收数据，返回值是一个字节对象，表示接收到的数据。
  • recvfrom(bufsize[, flags]) 从套接字接收数据，返回值是一对 (bytes, address)。
  • send(bytes[, flags]) 发送数据给套接字。
  • sendto(bytes, address) 发送数据给套接字，本套接字不应连接到远程套接字，而应由 address 指定目标套接字。
  • sendall(bytes[, flags]) 发送数据给套接字，本方法持续从 bytes 发送数据，直到所有数据都已发送或发生错误为止。成功后会返回 None。
  • setblocking(flag) 设置套接字为阻塞或非阻塞模式。
  • settimeout(value) 为阻塞套接字的操作设置超时。
  • setsockopt(level, optname, None, optlen: int)设置给定套接字选项的值。
  • getpeername() 返回套接字连接到的远程地址。
  • getsockname() 返回套接字本身的地址。
  • getsockopt(level, optname[, buflen]) 返回指定套接字选项的值。
  • gettimeout() 返回套接字操作相关的超时秒数（浮点数），未设置超时则返回 None。
  • dup() 创建套接字的副本。
  • shutdown(how) 关闭一半或全部的连接。
  • detach() 将套接字对象置于关闭状态，而底层的文件描述符实际并不关闭。返回该文件描述符，使其可以重新用于其他目的。
  • makefile(mode='r', buffering=None, *, encoding=None, errors=None, newline=None) 返回与套接字关联的 文件对象。
  • sendfile(file, offset=0, count=None) 使用高性能的 os.sendfile 发送文件，直到达到文件的 EOF 为止，返回已发送的字节总数。
  • fileno() 返回套接字的文件描述符（一个小整数），失败返回 -1。（配合 select.select() 使用）

# socket 模块的其它函数

提供与网络相关的服务，如 socket.setdefaulttimeout(timeout) 设置用于新套接字对象的默认超时。

# 套接字超时

一个套接字对象可以处于以下三种模式之一：阻塞、非阻塞或超时。套接字默认以阻塞模式创建，但是可以调用 setdefaulttimeout() 来更改。

connect() 操作受超时设置的约束，通常在调用 connect() 之前调用 settimeout()。（注意，无论 Python 套接字超时设置如何，系统网络栈都有可能返回自带的连接超时错误。）

accept() 方法返回的套接字行为取决于监听套接字的设置：

  • 如果监听套接字处于 阻塞模式 或 超时模式，则 accept() 返回的套接字处于 阻塞模式；
  • 如果监听套接字处于 非阻塞模式，那么 accept() 返回的套接字是阻塞还是非阻塞取决于操作系统。（如果要确保跨平台时的正确行为，建议手动覆盖此设置。）
