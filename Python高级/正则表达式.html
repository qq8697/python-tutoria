# 正则表达式

# 正则表达式语法

ps. 各语言基本通用。

  1. 匹配单个字符

    • . 任意字符，除了 \n
    • \d 数字
    • \D 非数字
    • \s 空白
    • \S 非空白
    • \w 字符 0-9a-zA-Z_
    • \W 非字符

  2. 匹配多个字符

    • * 0 或多次
    • + 1 或多次
    • ? 0 或 1 次
    • {m} m 次
    • {m,n} m 到 n 次
    • | 或
    • [...] 在其中的
    • [^...] 不在其中的

  3. 匹配开头和结尾

    • ^
    • $

  4. 分组

    • ()
    • \num引用分组
    • (?P<name>) 分组起别名
    • (?P=name) 引用别名

示例，邮箱匹配：

# 导入re模块
import re

# 使用match进行匹配
pattern = r"[a-zA-Z_0-9]{4,20}@(163|126)\.com$"
email = "nishixiuer@163.com"
result = re.match(pattern, email)

# 使用group提取数据
if result:
    print(result.group())  # nishixiuer@163.com
    print(result.group(1)) # 163

# 番外：贪婪与非贪婪

  • 贪婪： 匹配尽可能多的字符；
  • 非贪婪： 匹配尽可能少的字符。 Python 中默认是贪婪的，使用非贪婪操作符 ? 跟在 * + ? 后变为非贪婪模式。
import re

print(re.match(r"aa(\d+)","aa2343ddd").group(1))
print(re.match(r"aa(\d+?)","aa2343ddd").group(1))
print(re.match(r"aa(\d+)ddd","aa2343ddd").group(1))
print(re.match(r"aa(\d+?)ddd","aa2343ddd").group(1))
