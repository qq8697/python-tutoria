# 面向对象

面向对象的一些术语：对象、类、实例、方法（实例方法、类方法、静态方法）、属性（实例属性、类属性）、继承、多态、方法重写等。

# 定义类和创建实例对象

class Cat:
	def __init__(self, name):
		self.name = name
	def eat(self):
		print('%s is eating' % self.name)

tom = Cat("Tom")
tom.eat()
print(tom) # <__main__.Cat object at 0x0055C470>

  2. 初始化，为对象的属性设置初始值 -> __init__。

# 实例属性和实例方法

  • 实例的属性是实例对象独有的，实例的方法是对类的同名函数的引用（所以也叫类的方法），是该类所有实例共享的；
  • 实例初始化时会自动调用 __init__ 方法，在其内部为实例属性赋值；
  • 实例的 __dict__ 属性可以查看该实例的所有属性；
  • 实例方法同普通的函数一样，使用 def 定义，不同之处在于必须有一个额外的参数，按照惯例是 self，在函数中代表实例对象。

# 封装的意义

【函数（方法）定位更方便】：对数据处理时需要定位到使用哪个函数，使用面向对象编程，因为数据已经封装到对象这个独立的空间，且通过实例的 __class__ 属性能够找到类这一模板，模板内定义的处理该数据的方法较面向过程编程更容易定位。
【数据（属性）管理更方便】：全局变量是只能有一份的，面向过程中多个函数需要多个备份时需要利用其它变量来进行存储，面向对象通过封装会把变量变为对象的“全局变量”即属性，不同的对象可以有同名的“全局变量”。

# 类属性、类方法和静态方法

Python 中一切皆对象（实例的 __class__ 属性是对该类对象的引用），类同样也是拥有自己的属性和方法，同样会被加载到内存中且只有一份，通过 类名.属性名 和 类名.方法名() 直接访问。

类属性：

  1. 类属性用来记录与这个类相关的特征，如该类实例的个数；
  2. 属性向上查找机制：首先在对象内部查找对象的实例属性，没有找到会向上寻找类属性，所以 实例.类属性 也可以访问类属性，但不推荐；
  3. 实例.类属性名 = 值 赋值时只会给实例对象动态添加一个属性，不会影响到类属性的值；

类方法：
使用 @classmethod 装饰器标识这是一个类方法，cls 参数和实例方法的 self 参数类似，是惯用名称，指代类对象。 参见 >> 装饰器

class Cat:
	cat_count = 0
	
	def __init__(self, name):
		self.name = name
		Cat.cat_count += 1
	
	def eat(self):
		print('%s is eating' % self.name)

	@classmethod
	def show_cat_count(cls):
		print('there are %s cats' % cls.cat_count)

tom = Cat("Tom")
tommy = Cat("Tommy")
Cat.show_cat_count()

print(Cat.cat_count, Cat.eat)
Cat.eat(tommy)

静态方法：
一般在即不访问实例属性，也不访问类属性时使用，所以无参数，用 @staticmethod 装饰器标识，通过 类名.方法名() 调用。

class Cat:

	def __init__(self, name):
		self.name = name

	def eat(self):
		print('%s is eating' % self.name)

	@staticmethod
	def desc():
		print('cats are our friends')

Cat.desc()

小结：实例方法、类方法和静态方法

  1. 实例方法：方法内部需要访问实例属性和类属性，使用 self.实例属性 和 类名.类属性 分别访问；
  2. 类方法：方法内部只需要访问类属性，使用 cls.类属性 访问；（用于修改类属性）
  3. 静态方法：方法内部不需要访问实例属性和类属性；（用于和类对象与实例对象均无关且又不适合定义在类外让其他类共用的函数)

# 小结：对象、属性和方法

关于类对象、实例对象与类属性、类方法、实例属性、实例方法、静态方法总结如下：

  1. 类在内存中会分配一个空间，即类对象，包含类属性和所有方法（类方法、实例方法和静态方法）；
  2. 实例在内存中也会分配一个空间，即实例对象，包含实例属性；（其中 __class__ 属性指向类对象）
  3. 实例对象的属性是实例对象独有的，方法是共享自类对象的，在内存中只保存一份；
  4. 类对象只能调用类方法和静态方法；
  5. 实例对象可以调用类方法、实例方法和静态方法。
class Foo(object):
	def __init__(self):
		pass

	def ins_f(self):
		pass

	@classmethod
	def cls_f(cls):
		pass

	@staticmethod
	def stc_f():
		pass

f = Foo()

f.ins_f()
f.cls_f()
f.stc_f()

Foo.cls_f()
Foo.stc_f()
